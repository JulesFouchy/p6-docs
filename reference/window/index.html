read_pixel()#

Color read_pixel(glm::vec2 position) const;

Returns the color of the pixel at the given position.

The coordinates are expressed in the usual p6 coordinate system. The pixel is read from the current render target (which will be the screen in most cases, unless you used render_to_canvas())
